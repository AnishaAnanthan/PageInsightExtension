# Page Insight Extension

Page Insight is a simple Chrome extension that helps you understand any webpage faster. Instead of reading long articles or pages, you can click one button and instantly get a short summary, the overall sentiment, and the type of page you’re viewing.

## What it does

* Reads visible content from the current webpage
* Tells you what kind of page it is (article, blog, documentation, etc.)
* Analyzes the sentiment of the content
* Generates a clear and concise summary

## Why it’s useful

Perfect for quickly understanding articles, blogs, or documentation without spending too much time reading everything.

## Built with

* HTML, CSS, JavaScript
* Chrome Extensions API
* OpenAI API

## How to use

1. Open any webpage
2. Click the Page Insight extension
3. Press **Analyze Page**
4. View the page type, sentiment, and summary instantly

## Security note

The API key is not included in this repository for security reasons. Add your own key before running the extension.

## Limitations

* Works best on pages that are fully loaded
* Some highly dynamic or protected sites may return limited content
